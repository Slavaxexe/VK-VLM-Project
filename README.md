# Русскоязычная VLM для ответов на вопросы по изображениям

Учебный проект по дообучению готовой визуально-языковой модели на открытых данных VK.

## Идея

Модель получает изображение и вопрос на русском языке, затем формирует ответ. Вместо обучения VLM с нуля применяется QLoRA — параметро-эффективное дообучение готовой открытой модели. Такой подход позволяет провести эксперимент на доступном GPU и сохранить воспроизводимость результата.

Основной вопрос исследования: **улучшает ли русскоязычное instruction-tuning качество ответов модели на независимых бенчмарках GQA-ru и MMBench-ru?**

## Данные VK

| Назначение | Набор данных | Использование |
| --- | --- | --- |
| Обучение | [deepvk/LLaVA-Instruct-ru](https://huggingface.co/datasets/deepvk/LLaVA-Instruct-ru) | Русскоязычные диалоги по изображениям; из train будет сформирована обучающая выборка. |
| Оценка | [deepvk/GQA-ru](https://huggingface.co/datasets/deepvk/GQA-ru) | Независимая оценка ответов на визуальные вопросы. В обучение не входит. |
| Оценка | [deepvk/MMBench-ru](https://huggingface.co/datasets/deepvk/MMBench-ru) | Независимая оценка заданий с вариантами ответа. В обучение не входит. |

Все наборы опубликованы в [коллекции DeepVK Vision-Language Modeling](https://huggingface.co/collections/deepvk/vision-language-modeling).

## Структура

```text
notebooks/   последовательные ноутбуки исследования и обучения
docs/        паспорт проекта, итоговый отчёт и model card
results/     таблицы метрик, графики и качественные примеры
```

В репозитории не хранятся исходные датасеты и веса модели. Крупные результаты обучения будут версионироваться через DVC и DagsHub.

## План ноутбуков

1. `01_project_and_data_analysis.ipynb` — постановка задачи и исследовательский анализ данных VK.
2. `02_baseline_evaluation.ipynb` — исходная модель и её метрики до дообучения.
3. `03_qlora_finetuning.ipynb` — подготовка данных и QLoRA-дообучение.
4. `04_final_evaluation_and_examples.ipynb` — сравнение результатов и анализ примеров.

Подробная постановка приведена в [паспорте проекта](docs/project_passport.md).
